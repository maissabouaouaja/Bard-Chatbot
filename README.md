
Dans un marché du travail en évolution constante, recruter efficacement est devenu un défi majeur pour les entreprises. Pour relever ce défi, de plus en plus d’entreprises se tournent vers l’intelligence artificielle (IA), notamment pour optimiser le processus de re- crutement en intégrant des technologies innovantes dans les plateformes de gestion de CV. 
L’IA permet aux recruteurs d’analyser les CV de manière approfondie et rapide, en ex- trayant les informations clés telles que les compétences et l’expérience. Cela améliore la correspondance entre les profils des candidats et les exigences des postes, réduisant ainsi les biais humains et favorisant la diversité et l’inclusion. 
Pour parvenir à ces objectifs, nous avons commencé par collecter des CV au format PDF. Ensuite, nous avons utilisé des techniques de traitement du langage naturel (NLP) pour nettoyer les textes, extraire les mots-clés et normaliser les données textuelles. Les données ainsi traitées, considérées comme les données finales, ont ensuite été utilisées comme entrée pour les algorithmes de machine learning : K-Nearest Neighbors (KNN), Support Vector Classifier (SVC) et Régression Logistique. Suite à cette comparaison, la Régression Logis- tique et le Support Vector Classifier (SVC) ont démontré une précision supérieure dans la sélection des candidats. 
Mots clés : IA, Recrutement, KNN, SVC, Régression Logistique, NLP.
